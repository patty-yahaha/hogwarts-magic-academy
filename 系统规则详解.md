## 🪙 金加隆获取规则

### 初始配置
- **初始金加隆**: 100 枚
- **数据存储**: localStorage 中的 `hogwartsCoins`（当前金币）和 `cumulativeCoins`（累计金币）
- **数据同步**: 支持 Supabase 云端同步（登录用户）

### 获取方式

#### 🌍 世界语言学习区
| 活动 | 奖励 |
|------|------|
| ⏱️ 学习单词卡片满15分钟 | +30 金加隆 |
| 📝 测验得分 60-70分 | +1 金加隆 |
| 📝 测验得分 70-80分 | +3 金加隆 |
| 📝 测验得分 80-90分 | +20 金加隆 |
| 📝 测验得分 90-100分 | +50 金加隆 |

#### 🏰 霍格沃茨课程奖励
| 课程 | 奖励规则 |
|------|----------|
| 🧹 飞行课 | 每5根柱子 +3 金加隆 |
| ⚗️ 魔药课初级 | 每10个成功 +20 金加隆 |
| ⚗️ 魔药课中级 | 每10个成功 +30 金加隆 |

### 金币管理
- **累计金币**: 用于计算等级，消费不影响等级
- **当前金币**: 可用于购买物品和恢复精力
- **数据同步**: 支持本地存储和云端数据库同步

## ⚡ 精力值系统规则

### 什么是精力值？
精力值是玩家在魔法世界的行动力。每次进行课程或活动都会消耗精力值，精力值耗尽后需要通过购买食品恢复才能继续。

### 精力值基础规则
| 规则 | 详情 |
|------|------|
| ⏰ 初始精力 | 60点 |
| 📊 最大值 | 100点 |
| 🍔 恢复方式 | 三把扫帚商店购买食品恢复 |
| ⏳ 自然衰减 | 每分钟减少1点精力值 |
| 🌙 每日重置 | 每天重置到60点 |
| ⚠️ 最低要求 | 需要≥10点才能参加课程 |

### 精力值衰减机制
- **衰减速率**: 每分钟减少1点
- **离线暂停**: 玩家离线超过15分钟后停止衰减
- **操作追踪**: 通过点击、鼠标移动等操作更新最后活跃时间
- **数据存储**: `playerEnergy`（当前精力）、`playerLastActiveTime`（最后活跃）、`playerEnergyResetDate`（重置日期）

### 精力值恢复食品（三把扫帚）

| 巧克力蛋糕 | 🎂 | +40 | 70金加隆 |

| 苹果派 | 🥧 | +35 | 60金加隆 |

### 精力值显示状态
| 精力值 | 状态 | 提示 |
|--------|------|------|

| 20-49 | 正常 | 🟡 橙色 |

## 📬 邮箱系统规则

### 关于霍格沃茨猫头鹰邮件
猫头鹰会不定期给你送来各种邮件，包括：
- 🎁 **物品奖励** - 房间装饰品、背包物品
- 🔴 **记忆球** - 稀有特殊物品
- 📢 **重要通知** - 学院公告
- ✉️ **魔法事件** - 随机奇遇
- 🏆 **成就解锁** - 达成里程碑通知
- 💌 **导师邮件** - 好感度≥30时收到

### 邮件规则

#### 欢迎邮件
- **触发条件**: 首次访问（`testLetterSent` 未设置）
- **内容**: 随机房间装饰品
- **延迟**: 2秒后显示

#### 随机邮件
- **频率**: 每周最多3封
- **概率检查**: 页面加载5秒后检查
- **存储键**: `hogwartsLetters`

#### 导师奖励邮件
- **触发条件**: 导师好感度≥30
- **内容**: 个性化信件 + 随机小零食
- **每位导师仅发送一次**

### 邮件通知
- **徽章显示**: 邮箱按钮上显示未读数量
- **弹窗通知**: 新邮件到达时右下角显示5秒通知
- **点击通知**: 直接打开邮箱

## 💡 省力小技巧

1. **合理安排游戏时间**：精力值快用完时选择学习单词卡片，既不消耗精力又能积累学习时间！
2. **提前储备恢复食品**：记得在三把扫帚提前买好恢复食品！
3. **离线暂停衰减**：如果需要暂停游戏15分钟以上，精力值会自动停止衰减！
4. **每日重置**：每天登录会重置精力值到60点！
5. **导师好感度**：保持学习可以同时提升导师好感度，获得额外奖励邮件！
