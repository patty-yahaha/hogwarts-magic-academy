## 🪙 金加隆获取规则

### 初始配置
- **初始金加隆**: 100 枚
- **数据存储**: localStorage 中的 `hogwartsCoins`（当前金币）和 `cumulativeCoins`（累计金币）
- **数据同步**: 支持 Supabase 云端同步（登录用户）

### 金币管理
- **累计金币**: 用于计算等级，消费不影响等级
- **当前金币**: 可用于购买物品和恢复精力
- **数据同步**: 支持本地存储和云端数据库同步

## ⚡ 精力值系统规则

### 什么是精力值？
精力值是玩家在魔法世界的行动力。每次进行课程或活动都会消耗精力值，精力值耗尽后需要通过购买食品恢复才能继续。

### 精力值基础规则
| 规则 | 详情 |
|------|------|
| ⏰ 初始精力 | 60点 |
| 📊 最大值 | 100点 |
| 🍔 恢复方式 | 三把扫帚商店购买食品恢复 |
| ⏳ 自然衰减 | 每分钟减少1点精力值 |
| 🌙 每日重置 | 每天重置到60点 |
| ⚠️ 最低要求 | 需要≥10点才能参加课程 |

### 精力值衰减机制
- **衰减速率**: 每分钟减少1点
- **离线暂停**: 玩家离线超过15分钟后停止衰减
- **操作追踪**: 通过点击、鼠标移动等操作更新最后活跃时间
- **数据存储**: `playerEnergy`（当前精力）、`playerLastActiveTime`（最后活跃）、`playerEnergyResetDate`（重置日期）

- **触发条件**: 首次访问（`testLetterSent` 未设置）
- **内容**: 随机房间装饰品

#### 随机邮件
- **频率**: 每周最多3封
- **概率检查**: 页面加载5秒后检查
- **存储键**: `hogwartsLetters`

- **触发条件**: 导师好感度≥30
- **内容**: 个性化信件 + 随机小零食
- **每位导师仅发送一次**

- **徽章显示**: 邮箱按钮上显示未读数量
- **弹窗通知**: 新邮件到达时右下角显示5秒通知

## 🧹 飞行课

### 🎮 玩法规则
- **游戏类型**: 倒计时跑酷类
- **操作方式**: 按住空格键或点击屏幕控制巫师上下移动
- **目标**: 在90秒内通过尽可能多的柱子
- **生命值**: 3条生命，每撞到柱子掉1条，游戏不结束直到90秒结束或3条生命耗尽
- **红心显示**:
  - 剩2条命时：2颗红心忽隐忽现
  - 剩1条命时：1颗红心忽隐忽现
- **无敌机制**: 碰撞后会获得1.5秒无敌时间，巫师会闪烁提示
- **视觉效果**: 根据速度变化有不同的背景流动效果和柱子颜色
- **胜利条件**: 90秒倒计时结束或生命值耗尽
- **柱子生成**: 从右向左无限生成，空间距离固定不变

### 🪄 飞天扫帚系统
- **可用扫帚**:
  - 🧹 普通飞天扫帚：无加成
  - 🌟 彗星260：价格150金币，速度+10%，缝隙+8px，金币+15%
  - 💫 火弩箭11号：价格200金币，速度+15%，缝隙+15px，金币+5%
  - ✨ 光轮2001：价格300金币，速度+20%，缝隙+10px，金币+10%
  - 🔥 火弩箭：价格500金币，速度+30%，缝隙+20px，金币+20%，好感度+1
- **扫帚选择**: 游戏开始前可以选择拥有的扫帚
- **属性加成**: 速度、缝隙大小、金币获取、好感度增长都有加成
- **购买系统**: 扫帚需要用金币购买，保存在localStorage中

### ⏱️ 倒计时规则
- **每局时长**: 90秒
- **视觉显示**: 顶部中央显示倒计时
- **结束条件**: 时间为0或3条生命耗尽
- **每日限制**: 每日游戏次数和游戏时间都有统计
- **时间限制**: 每日最多60分钟游戏时间

### 🏃‍♂️ 速度机制
- **初始速度**: 3像素/帧
- **速度增长**: 每穿过1根柱子速度+0.05
- **最高速度**: 12像素/帧（速度上限）
- **空间距离**: 360像素固定（无论速度多少，屏幕上柱子间距不变）
- **生成方式**: 根据累计移动距离生成柱子，确保视觉间隔一致
- **速度区间效果**:
  - **3-4**: 缓慢飘动，普通木色，轻柔风声，提示"你刚骑上扫帚~"
  - **4-6**: 正常流动，木色带微光，风声稍强，提示"飞得不错！"
  - **6-8**: 明显加速，银色闪光，呼啸风声，提示"哇，越来越快了！"
  - **8-10**: 快速后退，金色发光，尖锐风声，提示"你是专业的找球手！"
  - **10-12**: 残影效果，彩色炫光，刺耳风声+心跳声，提示"天哪，这速度破纪录了！"

### 🪙 金加隆规则
- **实时显示**: 玩家头像旁边实时显示当前金币数
- **基础奖励**: 根据当前速度决定每根柱子的金币：
  - 速度 < 5: +1金币/根
  - 5 ≤ 速度 < 8: +2金币/根
  - 8 ≤ 速度 < 11: +3金币/根
  - 速度 ≥ 11: +4金币/根
- **连击奖励**: 连续穿过柱子获得额外奖励（撞到柱子重置）：
  - 10连击：+5金币
  - 20连击：+15金币
  - 30连击：+30金币
  - 连击时播放特殊音效
- **扫帚加成**: 使用高级扫帚可以获得金币百分比加成（10%-20%）
- **防刷机制**:
  - 每日前3局：全额奖励
  - 第4局及之后：50%奖励
- **数字提示**: 获得金币时有短暂的数字提示显示
- **金币系统**: 与全局CoinManager同步，支持云端存储

### ❤️ 好感度规则
- **导师**: 霍琦夫人 (Madam Hooch)
- **实时显示**: 霍琦夫人头像旁边实时显示当前好感度
- **数据存储**: 统一存储在 `hogwarts_mentor_affection['hooch']` 中
- **保护机制**: 好感度最低为0（不会为负数）
- **增加条件**:
  - 当日第一次游戏：+10（显示"欢迎你！"提示）
  - 每穿过1根柱子：+0.2（显示"不错！"提示）
  - 使用火弩箭扫帚：好感度增长+1
- **减少条件**:
  - 撞到柱子（第一次）：-0.1（显示"啊哦！"提示）
  - 撞到柱子（连续2次）：-0.3（显示"集中注意力！"提示）
  - 一局内3条命用完（死亡）：-1.0（显示"下次要更小心哦"提示）
- **数字提示**: 好感度变化时有短暂的数字提示显示

### 📊 游戏结束总结
- **穿过柱子数**: 显示本次游戏穿过的柱子总数
- **获得金币**: 显示本次游戏获得的金加隆数量
- **霍琦夫人好感度**: 显示最终的好感度
- **新纪录**: 如果穿过柱子数超过历史最高，会标记为"新纪录!"
- **再来一次**: 游戏结束后可以选择"再来一次"或"返回"

## ⚗️ 魔药课

### 🎮 玩法规则
- **游戏类型**: 记忆与匹配类
- **操作方式**: 点击材料、输入数量，放入坩埚
- **目标**: 根据配方准确调制魔法药水
- **难度级别**:
  - 初级 (Beginner): 2种材料，基础配方
  - 中级 (Intermediate): 4种材料，复杂配方
- **失败条件**: 配方错误或超时
- **胜利条件**: 连续成功完成多个配方

### 🪙 金加隆规则
- **初级课程**: 每成功10个配方奖励20金币
- **中级课程**: 每成功10个配方奖励30金币
- **连胜奖励**: 连续成功有额外金币加成
- **失败惩罚**: 错误配方会扣除少量金币
- **最高奖励**: 完成全部课程可获得约100金币

### ❤️ 好感度规则
- **导师**: 斯内普教授 (Professor Snape)
- **增加条件**:
  - 每连续成功5个配方增加1点好感度
  - 中级课程有额外好感度奖励
  - 使用正确材料顺序有加成
- **减少条件**: 每累计失败10次减少1点好感度
- **数据存储**: 存储在 `potionsGameState.favorability` 中

## 🛡️ 黑魔法防御术

### 🎮 玩法规则
- **游戏类型**: QTE反应类
- **操作方式**: 按屏幕提示依次按键防御黑魔法攻击
- **目标**: 正确防御每轮的黑魔法攻击
- **攻击类型**: 摄魂怪、博格特、恶咒、红帽子、格林迪洛等13种攻击类型
- **生命值**: 10格，每3次错误或超时扣1格血
- **攻击机会**: 每个攻击有3次按键机会
- **按键提示**: 显示3秒后消失
- **胜利条件**: 完成3轮训练

### 🪙 金加隆规则
- **基础奖励**: 完成3轮可获得30金币
- **轮次奖励**:
  - Round 1 (基础): 10秒/攻击，完成获得5金币
  - Round 2 (进阶): 5秒/攻击，完成获得10金币
  - Round 3 (大师): 2秒/攻击，完成获得15金币
- **总奖励**: 完美完成所有3轮可获得30金币
- **金币系统**: 与全局CoinManager同步，支持云端存储

### ❤️ 好感度规则
- **导师**: 卢平教授 (Professor Lupin)
- **数据存储**: 统一存储在 `hogwarts_mentor_affection['lupin']` 中
- **实时对话**: 每轮开始和攻击时会有语音提示
- **胜利反馈**: 成功防御时显示"呼神护卫"等咒语特效
- **失败反馈**: 防御失败时显示生命值减少动画
- **成功反馈**:
  - 成功防御攻击: "很好，反应迅速！"
  - 连续防御成功: "出色的防御技巧！"
  - 完美完成一轮: "太棒了，你已经掌握了防御术的精髓！"
- **失败反馈**:
  - 防御失败: "小心，集中注意力！"
  - 连续失败: "别灰心，多练习就能提高反应速度！"
  - 生命值耗尽: "防御术需要更多练习，下次加油！"
- **好感度变化**:
  - 连续 3 次防御都成功→ +5 好感度
  - 完成全部 3 轮训练（通关胜利）→ +20 好感度
  - 一轮内 0 错误、0 超时（完美一轮）→ 额外 +10 好感度
  - 防御错误 1 次→ -1 好感度
  - 防御超时 1 次→ -1 好感度
  - 挑战失败（生命值归 0）→ -15 好感度

### 🎯 游戏流程
1. **游戏开始**: 显示玩法介绍，点击"开始训练"
2. **Round 1 (基础)**: 3个按键的攻击，10秒/攻击
3. **Round 2 (进阶)**: 5个按键的攻击，5秒/攻击
4. **Round 3 (大师)**: 4个按键的攻击，2秒/攻击
5. **游戏完成**: 显示总金币奖励和胜利动画

### 📊 界面元素
- **玩家区域**: 显示玩家头像和生命值
- **老师区域**: 卢平教授头像和实时对话
- **攻击显示**: 当前攻击的名称和表情
- **按键提示**: 显示需要按下的按键序列
- **计时器**: 显示攻击剩余时间
- **金币奖励**: 显示每轮获得的金币
- **游戏结束**: 显示完成面板和下一轮按钮

### ⌨️ 攻击类型示例
- **摄魂怪**: W + 空格 + D
- **博格特**: ↓ + ↑ + ← + → + 回车
- **恶咒**: ← → ← →
- **阿瓦达索命**: A + S + J + K
- **钻心咒**: ↑ + ↓ + W + S
- **火焰风暴**: W + A + S + D + 空格

- **操作方式**: 根据咒语效果提示，在输入框中拼写正确的英文咒语
