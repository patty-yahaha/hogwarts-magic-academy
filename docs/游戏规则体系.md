## ⚡ 精力值系统

### 什么是精力值？
精力值是玩家在魔法世界的行动力。每次进行课程或活动都会消耗精力值，精力值耗尽后需要通过购买食品恢复才能继续。

### 精力值规则
| 规则 | 详情 |
|------|------|
| ⏰ 初始精力 | 100点 |
| 🍺 恢复方式 | 只能通过三把扫帚商店购买食品恢复 |
| 🍔 恢复规则 | 不同食品恢复不同数量的精力值 |
| ⏳ 自然衰减 | 精力值会随游玩时间不断减少 |
| 🌙 每日重置 | 每天精力值会在0点重置 |

### 💡 省力小技巧
合理安排游戏时间，精力值快用完时选择学习单词卡片，既不消耗精力又能积累学习时间！记得提前储备一些恢复食品！

## ❤️ 好感度系统

### 💕 好感度获取规则
| 活动 | 规则 |
|------|------|
| 🧹 飞行课 | 每连续通过10根柱子 → 好感度 +1 |
| 🧹 飞行课 | 每累计5次通过柱子少于1根 → 好感度 -1 |
| ⚗️ 魔药课 | 每连续成功5个 → 好感度 +1 |
| ⚗️ 魔药课 | 每累计失败10次 → 好感度 -1 |

### 🌟 好感度有什么用？
好感度会影响角色在聊天中的回复态度。好感度越高，角色越友好！解锁特殊剧情需要达到一定好感度。

## 📬 邮箱系统

### 📮 关于霍格沃茨猫头鹰邮件
猫头鹰会不定期给你送来各种邮件，包括：
- 🎁 **物品奖励** - 房间装饰品、背包物品
- 🔴 **记忆球** - 稀有特殊物品
- 📢 **重要通知** - 学院公告
- ✉️ **魔法事件** - 随机奇遇
- 🏆 **成就解锁** - 达成里程碑通知

### 📅 邮件规则
- 📮 每天最多收到 3 封随机邮件
- 🎁 有 10% 概率收到特殊物品邮件
- 🔴 记忆球邮件有 10% 概率出现
- 📬 未读邮件会在头像上显示数量
